# school_app

一个基于微信小程序的学院app

![](https://github.com/fruittt/school_app/blob/master/image/screenshot.gif?raw=true)

## 基本功能

1.借书查询功能

2.馆藏查询功能

3.RGB与十六进制转换

## 暂缺

1.数据库读取写入

2.界面UI美化

## 使用

1. 克隆此项目到本地

2. 在微信开发工具中创建项目

3. 选择该项目目录
